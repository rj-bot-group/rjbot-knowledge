# RJ-BOT 知识库项目

## 项目简介
这是从 RJ-BOT 官网（https://www.rj-bot.com/）爬取的知识库，包含公司介绍、产品信息、解决方案等内容。

## 知识库结构

```
docs/
├── Index.md                          # 知识库索引
├── About/                            # 关于我们
├── Contact/                           # 联系我们
├── FAQ/                               # 常见问题
├── Home/                              # 主页
├── News/                              # 新闻动态
└── Products/                          # 产品中心
```

## 文档格式
所有文档使用 Obsidian Markdown 格式，包含：
- 页面标题
- 源 URL
- 爬取时间
- 分类内容
